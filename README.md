# boletin

NSK Fashion Website
NSK Fashion Logo
A sleek, responsive single-page website for NSK Fashion, showcasing bold, artistic clothing collections with a dark, edgy aesthetic. Built with pure HTML, CSS, and JavaScript for a lightweight, fast-loading experience.
Features

Hero Header: Eye-catching logo, slogan, and carousel of featured looks.
Featured Products: Spotlight on weekly highlights with prices and CTAs.
Creative Story: Narrative section with imagery and philosophy.
Collections Gallery: Grid of new designs with hover effects.
Community Section: Social links, newsletter signup, and hashtag sharing.
Footer: Quick links, customer service, and info.
Responsive Design: Mobile-friendly with media queries.
Smooth Carousel: JS-powered slider with auto-advance.

Tech Stack

HTML5 for structure
CSS3 with variables, grids, and transitions
Vanilla JavaScript for interactivity

Setup

Clone the repo: git clone <repo-url>
Open index.html in your browser.
Customize images and links as needed.
